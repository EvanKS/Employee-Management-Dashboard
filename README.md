# Employee Management Dashboard

## Project Overview
The **Employee Management Dashboard** is a frontend web application developed using **Angular and TypeScript**.  
The project focuses on building a clean, professional, and user-friendly dashboard to display and manage employee information in an organization.

This application demonstrates core Angular concepts such as **component-based architecture, data binding, structural directives, and UI theming** using Angular Material.  
The design is inspired by modern **job portals and HR management systems**, ensuring a real-world look and feel.

---

## Project Objectives
- To understand and implement **Angular components** and modular design
- To apply **TypeScript models and interfaces** for structured data
- To demonstrate **data binding and event handling**
- To build a **professional dashboard UI** using Angular Material
- To improve UI/UX using hover effects, active navigation styling, and theming

## How to Run the Project

1. Clone the repository
git clone <repository-url>

2. Navigate to the project folder
cd <project-folder-name>

3. Install dependencies
npm install

4. Start the development server
ng serve

5. Open in browser
http://localhost:4200

The application will automatically reload when you make changes.
